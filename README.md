> 100% 천연 탈취제를 활용한 B2B 악취제거 전문기업 — 미국 OMI사 에코솝(Ecosorb) 국내 독점 공급사

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 회사명 | (주)냄새는 못 참지 Co., Ltd. |
| 사이트 목적 | 악취제거 B2B 사업 소개 및 제품 홍보 |
| 디자인 테마 | 깨끗하고 산뜻한 그린 계열 (Green + White) |

---

## ✅ 완성된 기능

### 페이지 구성
- **index.html** — 메인 홈 + 인사말 (회사소개 > 인사말)
- **location.html** — 오시는길 (카카오맵 임베드, 연락처, 교통 안내)
- **business.html** — 사업소개 (개요, 제거 가능 악취, 주요 고객, 서비스 유형, 투자비용, 마케팅 전략, 성공 핵심요소)
- **products.html** — 제품소개 (탭 필터, 검색, 제품 상세 모달)

### 공통 기능
- ✅ 상단 고정 헤더 + 드롭다운 네비게이션
- ✅ 모바일 햄버거 메뉴
- ✅ 푸터 (회사 정보, 링크 맵)

### 제품소개 특화 기능
- ✅ **카테고리 탭 필터** (전체 / 천연 탈취제 / 화장실 탈취제 / 녹조곰팡이 제거 / 기타 제품)
- ✅ **실시간 키워드 검색**
- ✅ **제품 상세 모달** (팝업) — 자세히 보기 클릭 시
- ✅ 탭별 제품 수 카운트 동적 표시
